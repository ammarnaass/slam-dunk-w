# كيفية استخدام سكريبت screenshots التلقائي

## 1. تثبيت المكتبات المطلوبة

```bash
npm install puppeteer --save-dev
```

## 2. إنشاء مجلد الصور

```bash
mkdir public
mkdir public/thumbnails
```

## 3. تشغيل السكريبت

```bash
node get-mega-screenshots.js
```

⏱️ **سيستغرق حوالي 10-15 دقيقة لـ 101 حلقة**

## 4. بعد الانتهاء

السكريبت سيقوم بـ:
- فتح كل رابط Mega
- أخذ screenshot من الفيديو
- حفظ الصورة في `public/thumbnails/`
- تحديث `episodes.json` تلقائياً

## ملاحظات مهمة

- تأكد من اتصالك بالإنترنت
- لا تغلق النافذة حتى ينتهي
- إذا فشل رابط، سيتم تخطيه
- يمكنك إيقاف السكريبت بـ Ctrl+C

## البدائل الأسرع

إذا كنت تريد نتيجة فورية:

### أ. استخدام صورة شعار موحدة:
```bash
node use-single-logo.js
```

### ب. استخدام الصورة من iframe فقط:
الموقع سيستخدم صورة Mega الافتراضية (poster) تلقائياً
